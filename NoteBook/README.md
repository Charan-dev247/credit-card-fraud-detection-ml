# Credit Card Fraud Detection using Machine Learning

## Overview
This project focuses on detecting fraudulent credit card transactions using machine learning techniques. Fraud detection is a challenging problem because fraudulent transactions are extremely rare compared to normal transactions.

## Dataset
The dataset contains transactions made by European cardholders in September 2013.

- Total transactions: 284,807
- Fraud transactions: 492
- Fraud percentage: 0.17%

## Problem
The dataset is highly imbalanced, making fraud detection a difficult classification problem.

## Methods Used
- Logistic Regression
- Random Forest

## Evaluation Metrics
Since the dataset is imbalanced, the following metrics are used:

- Precision
- Recall
- F1-score
- ROC-AUC

## Results
Random Forest performed better than Logistic Regression for detecting fraudulent transactions.

## Future Improvements
Possible improvements include:

- Using SMOTE for handling class imbalance
- Trying gradient boosting algorithms
- Deep learning approaches
